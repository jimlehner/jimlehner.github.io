# Jim Lehner - Personal Website (Hugo)

This is the source code for my personal website built with [Hugo](https://gohugo.io/) and deployed to GitHub Pages.

Live site: https://jimlehner.github.io.

---

## Tech Stack

- Static site generator: Hugo (extended version)
- Hosting: GitHub Pages
- CI/CD: GitHub Actions
- Styling: Hugo theme (see below)

## Project Structure
```
├── archetypes/
├── assets/
├── content/        # Pages and blog posts
├── layouts/        # Custom layouts (if used)
├── public/
├── resources/
├── static/         # Images, CSS, JS
├── hugo.toml      # Site configuration
├── README.md
└── .github/
    └── workflows/  # GitHub Actions deployment
```

## Local Development

Run the Hugo development server using Powershell:

```hugo serve```

Then open:
http://localhost:1313/
