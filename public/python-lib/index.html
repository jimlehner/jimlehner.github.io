process_improvement

A Python library dedicated to process improvement

process_improvement.py is a Python library that serves as a free alternative to paid tools like Excel, Minitab, and JMP. Assuming a rudimentary knowledge of the Python programming language, the library simplifies the task of performing calculations and generating the figures that are essential for understanding variation, reducing costs, and improving quality.

Although the primary tool of the library is the XmR chart, the most useful type of process behavior chart, it also generates capability histograms, combo charts, trended X charts, limit charts, and a grid of X charts providing a system-level view of a process called network analysis.
